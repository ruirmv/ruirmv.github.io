Docker

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers. With docker we bundle our application, including all of its runtime dependencies into an image and share these images in a registry.

Download an image

docker pull image docker pull [registry/][namespace/]<repository>[:version/]

Running an image

docker run image

Creating an image

To create an image we must creat a Dockerfile that describes it.

# our image has to be based on a base image which contains the dependencies we need FROM base image # we can also copy files from our current directory to a location in the image COPY file location # we can also execute commands in the image is created RUN command

After this we need to build the actuall image.

docker build -t image name .
