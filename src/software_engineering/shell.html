Common tasks

Send task to background

When running a task, and move it to background, type CTRL+Z and then bg.

Alternatively, when running the command, type command &.

Keyboard Shortcuts

Moving the cursor

  • to the start of the line - CTRL + A
  • to the end of the line - CTRL + E
  • to the previous character - CTRL + B
  • to the next character - CTRL + F
  • to the previous word - ALT + B
  • to the next word - ALT + B

Deleting the text

  • deletes the character under the cursor - CTRL + D
  • removes the characters from the cursor to the end - CTRL + K
  • removes all the text from the cursor to the beginning of the line - CTRL + X and DEL
