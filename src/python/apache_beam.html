Best Practices

Typing

The output type of a beam.DoFn should be either a Generator (prefered) or an Iterator.

Returns vs. yields

A beam.DoFn should consistently use either return with a generator or yield individual elements. See documentation.

If there is nothing to return, an empty return or yield should be used.
