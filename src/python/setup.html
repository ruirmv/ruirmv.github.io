Setting up a project

Python Version

pyenv

Allows to install multiple Python versions on the system.

  1. # list the available python versions
  • pyenv install --list
  1. # install a specific version
  • pyenv install -v version
  1. # set local version, replace with global for global version
  • pyenv local version

Resources

  • How to use pyenv? Installation guide and commands.

Virtual Environments

venv

The Python version used on the virtual environment is the one used to create it.

  1. python3 -m venv environment_name
  2. source environment_name/bin/activate
  3. deactivate

Package Management

pip

Checking the package version

  1. pip show package

Checking instaled packages

  1. pip list

Get requirements.txt

  1. pip freeze --local > requirements.txt
