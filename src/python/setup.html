Setting up a project

Python Version

pyenv

Allows to install multiple Python versions on the system.

# list the available python versions pyenv install --list # install a specific version pyenv install -v version # set local version, replace with global for global version pyenv local version

Resources

  • How to use pyenv? Installation guide and commands.

Virtual Environments

venv

The Python version used on the virtual environment is the one used to create it.

python3 -m venv environment_name source environment_name/bin/activate deactivate

Package Management

pip

Checking the package version

pip show package

Checking instaled packages

pip list

Get requirements.txt

pip freeze --local > requirements.txt
