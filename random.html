Creating a Pipeline

  1. Create the pipeline with a run method.
  2. Create a pipeline_name_main.py.
  3. Add pipeline to ci/pipeline.py along with it's configurations.
  4. Create a metadata file for the pipeline.
  5. Add the pipeline to jenkinsfile and jenkinsfile_deploy.
  6. Run the christmas tree tests with GOLDEN_UPDATE=true as an environmental variable.
  7. Make the pull request.
  8. After the PR is accepted check Jenkins → Jobs → importer_dataflow_deploy → Blue Ocean, to check for the deployment of the pipeline template.
